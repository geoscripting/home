Resources¶

One week is short and there is so much more to discover. So if you’re not tired yet, take a look at the following projects.

Geo data processing packages¶

  • PySAL

  • PyGEOS

  • PyGEOS and GeoPandas - GitHub

Processing Big Data¶

  • Dask: Processing of large data → Tutorial

  • multiprocessing

  • Cython and Pandas

  • Numba: High Performance python compiler

(Geo) Data Visualization¶

  • dash: Interactive dashboards using Python

  • plotly-geo

  • plotly maps

Advanced Python Data structures¶

  • Interfaces, Mixins and Building Powerful Custom Data Structures in Python

Conferences related to Python Data Analysis¶

Some of these conference put the talks and workshops online. A good way to stay up to date on current Python developments.

  • SciPy

  • EuroSciPy

  • GeoPython

  • PyData

  • PyCon

  • .. and many more.

Books¶

Here are some book recommendations taken from [1]

The Python Programming Language¶

  • Mark Lutz (2013): Learning Python. (Safari Books)

  • Luciano Ramalho (2015): Fluent Python. (Safari Books)

  • Paul Gries, Jennifer Campbell, Jason Montojo (2013): Practical Programming: An Introduction to Computer Science Using Python 3. Pragmatic Bookshelf (Safari Books)

  • Allen B. Downey (2016): Think Python: How to Think Like a Computer Scientist (Free Textbook)

Scientific Computing with Python¶

  • Robert Johansson. Lectures on Scientific Computing with Python. (GitHub)

  • Anthony Scopatz and Kathryn D. Huff (2015): Effective Computation in Physics. (Safari Books)

  • Hans Petter Langtangen (2014): A Primer on Scientific Programming with Python. (eBook)

  • Harriet Dashnow, Juan Nunez-Iglesias, and Stéfan van der Walt (2017): Elegant SciPy: The Art of Scientific Python. (GitHub)

High Performance Computing with Python¶

  • Kurt W. Smith (2015): Cython. (Safari Books)

  • Micha Gorelick, Ian Ozsvald (2014): High Performance Python. (Safari Books)

  • Fernando Dogli (2015): Mastering Python High Performance. (Safari Books)

  • Schmidt, B., Gonzalez-Dominguez, J., Hundt, C., & Schlarb, M: (2017). Parallel programming: concepts and practice. Morgan Kaufmann. (Google Books)

[1] http://michael-kraus.org/high-performance-computing-with-python.html

Scientific papers¶

  • Wilson, G., Aruliah, D. A., Brown, C. T., Hong, N. P. C., Davis, M., Guy, R. T., … & Waugh, B. (2014). Best practices for scientific computing. PLoS biology, 12(1).

  • Wilson, G., Bryan, J., Cranston, K., Kitzes, J., Nederbragt, L., & Teal, T. K. (2017). Good enough practices in scientific computing. PLoS computational biology, 13(6).
