2. Vectorized Data Processing¶

The assignment can be accessed and submitted via GitHub classroom. → Accept Assignment 2: Vectorized Data Processing

This assignment consists of 3 Jupyter Notebooks. (The third one will follow). Work through all of them and answer the questions.

  • Part 1: Vectorized Computation using Numpy
  • Part 2: Vectorized Computation using Pandas and GeoPandas

Learning goals¶

  1. Explain the logical set-up of a computer.

  2. Name the factors affecting performance of a computer.

  3. Explain the von Neumann Bottleneck and strategies of how to overcome it.

  4. Use vectorization in Python based on numpy and (geo)pandas.

  5. Fork a Github repository.
